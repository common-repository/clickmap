== Description ==

Clickmap is a real-time heatmap tool to track mouse clicks and scroll paths of your website visitors. Gain intelligence about what's hot and what's not, and optimize your conversion with Clickmap.


== Installation ==

1. Upload all files to the `/wp-content/plugins/clickmap` directory (create it if necessary)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the 'Settings' menu in WordPress and enter your Clickmap Tracker Code ID. If you don't have a Clickmap account yet, sign up for free at www.getclickmap.com to get your Tracker Code.
